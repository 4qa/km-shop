Order List PENDING
===================

Получить список заказов со статусом PENDING

```shell title="Method <span class='color-method'>GET</span>"
/api/orders/pending
```

Варианты ответа

| Status code                          | Description    |
|--------------------------------------|----------------|
| <span class='color-200'>200</span>   | OK             |
| <span class='color-error'>500</span> | Internal Error |
