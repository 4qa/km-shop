Product Update
===================

Обновить данные продукта

```shell title="Method <span class='color-method'>PUT</span>"
/api/products/{id}
```

Варианты ответа

| Status code                          | Description    |
|--------------------------------------|----------------|
| <span class='color-200'>200</span>   | OK             |
| <span class='color-error'>500</span> | Internal Error |
